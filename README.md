# elite102-banking-app

A simple banking app built in Python that connects to a MySQL database. I built this as part of the Code2College Elite 102 program.
Features

Create a bank account
Deposit and withdraw money
Check your balance
View all accounts
Transaction history with timestamps

How to run
-------------------------------
Make sure you have Python and MySQL installed, 
Enter your MySQL login, then just run:
"python main.py" in the terminal

Notes
-----------------------------------
The app uses a command line interface — just pick a number from the menu and follow the prompts. All data is saved to a local MySQL database called BankingApplication.
